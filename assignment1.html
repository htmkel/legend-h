Introduction to HTML

HTML (Hypertext markup languange) is the foundation of web development, it is a markup language used to structure the content on the web pages.

CHARACTERISTICS OF HTML

1- It is a languange that can be easily understood and can be modified

2 -It provides a more flexible way to design web pages along with text

3 -links can be added to the web pages so it helps the reader to browse the information of their interests.

4 - you can display HTML documents on any platform such as windows and linux.

5 - graphic,videos and sounds can also be added to the web pages giving an attractive look.
